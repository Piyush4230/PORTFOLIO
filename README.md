## 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/Piyush4230/YOUR-PORTFOLIO-REPO.git
```

Go to the project directory

```bash
cd PORTFOLIO
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
